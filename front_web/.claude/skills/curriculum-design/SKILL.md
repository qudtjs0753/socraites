# 커리큘럼 설계 스킬

## 목적

JS 초중급 수준의 학습자가 프론트엔드 개발을 체계적으로 배울 수 있도록 맞춤형 학습 로드맵을 설계한다.

## 수준 파악 질문

학습자 정보가 없을 때, 다음 5개 질문으로 수준을 파악한다:

```
1. JS에서 어느 정도까지 알고 있나요?
   a) 변수, 함수, 조건문/반복문 정도
   b) 배열 메서드(map, filter), 화살표 함수, 구조 분해 할당
   c) Promise, async/await, 모듈 시스템(import/export)까지

2. 프론트엔드 프레임워크 경험이 있나요?
   a) 없음 (순수 HTML/CSS/JS만)
   b) 다른 프레임워크 경험 있음 (Vue, Angular 등)
   c) React 조금 해봤음

3. 학습 목표가 무엇인가요?
   a) 개념 이해 (취업/이직은 아직 아님)
   b) 포트폴리오용 프로젝트 1개 완성
   c) 취업/이직 준비

4. 주당 몇 시간 정도 학습할 수 있나요?
   a) 5시간 이하
   b) 5~15시간
   c) 15시간 이상

5. 특별히 먼저 배우고 싶은 것이 있나요? (선택)
```

## 수준별 시작 단계

**a 수준 (JS 기초)**: Stage 0 (JS 필수 개념 보강)부터 시작
**b 수준 (JS 중급)**: Stage 1 (React 기초)부터 바로 시작
**c 수준 (async/await 이해)**: Stage 1을 빠르게 진행, Stage 3부터 집중

## 로드맵 산출물 형식

```markdown
# 프론트엔드 학습 로드맵

**현재 수준:** [파악된 수준 요약]
**목표:** [학습자 목표]
**예상 기간:** [주당 학습 시간 기반 총 기간]

## 전체 단계

| 단계 | 주제 | 예상 기간 | 상태 |
|------|------|----------|------|
| Stage 0 | JS 필수 개념 보강 | 3~5일 | ⬜ |
| Stage 1 | React 기초 | 1~2주 | ⬜ |
| Stage 2 | React 심화 | 1주 | ⬜ |
| Stage 3 | Next.js 기초 | 1~2주 | ⬜ |
| Stage 4 | TanStack Query | 1주 | ⬜ |
| Stage 5 | Tailwind CSS | 3~4일 | ⬜ |
| Stage 6 | 개발 도구 | 2~3일 | ⬜ |
| Stage 7 | 테스트 코드 | 1주 | ⬜ |
| Stage 8 | 배포 | 2~3일 | ⬜ |

## 현재 단계: [Stage N] - [주제명]

### 이번 단계 목표
- [ ] [학습 목표 1]
- [ ] [학습 목표 2]
- [ ] [실습 과제]

### 이번 단계가 끝나면
[다음 단계와의 연결 설명]
```

## 각 Stage 상세 내용

`references/learning-phases.md` 참조 — 각 Stage의 핵심 개념, 실습 과제, 완료 기준이 포함되어 있다.

## 로드맵 업데이트 규칙

- 학습자가 단계를 완료했다고 보고하면 상태를 ✅로 업데이트한다.
- 특정 단계가 어렵다는 피드백이 오면 해당 단계를 더 작은 단위로 분해한다.
- 학습 속도가 예상보다 빠르면 다음 단계를 당긴다.
